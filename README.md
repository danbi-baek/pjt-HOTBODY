### HOT BODY

---

음식분류 AI와 복근 사진을 이용한 체지방 예측 AI를 이용해 식단관리를 해주는 웹 사이트


![캡처](./images/2.png)



##### 기능

1. 식단 업로드 시 이미지에 포함된 음식의 칼로리와 성분을 측정
2. 복부 사진 업로드 시 체지방률을 계산 후 인바디를 예상해 그래프로 제공
3. 체지방에 따른 식단 루틴 추천 (3분류 정도로 나눠서 만들어진 식단 이미지 랜덤으로 보여줌)
